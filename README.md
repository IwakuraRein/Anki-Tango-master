# ja_word_card

A small GUI tool for querying Japanese dictionary entries and formatting them as JSON or Anki-friendly text.

## Installation

Install the project from the repository root:

```
pip install .
```

For development, install it in editable mode:

```
pip install -e .
```

## Usage

Launch the GUI after installation:

```
ja_word_card
```

You can also run it as a Python module:

```
python -m ja_word_card
```
